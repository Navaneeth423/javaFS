# 📚 Java Learning Projects Repository

> This branch contains hands-on Java projects starting from beginner-level fundamentals and progressively moving toward advanced concepts and real-world applications.

---

## 🧠 Overview

This branch is dedicated to learning and practicing **Java programming** through structured projects.  
The projects evolve gradually — beginning with basic syntax and OOP fundamentals, and progressing towards advanced Java development including:

- Collections Framework  
- Exception Handling  
- Interfaces & Abstraction  
- Lambda Expressions & Functional Interfaces  
- Streams API  
- File Handling  
- JDBC  
- Mini applications

Each project builds on previous concepts to help strengthen understanding and confidence in Java.
